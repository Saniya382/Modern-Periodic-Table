# Modern Periodic Table (C Programming)

## Description
This project is a console-based application developed in C that represents a modern periodic table. It allows users to view and access information about chemical elements such as atomic number, symbol, and atomic mass in a structured format.

## Technologies Used
- C Programming Language

## Features
- Displays details of chemical elements  
- Structured representation of periodic table data  
- User-friendly console interface  
- Efficient data handling using basic C concepts  

## How to Run
1. Open the project in any C compiler (e.g., GCC, CodeBlocks, Turbo C)  
2. Compile the code  
3. Run the executable file  
4. Follow on-screen instructions to view element details  

## Key Learnings
- Understanding of C programming fundamentals  
- Use of structures and conditional statements  
- Improved logical thinking and problem-solving skills
